📡 About These Feeds
Waveform images are fetched server-side from the IRIS timeseriesplot API during each GitHub Actions run (twice daily). Each image shows a rolling 24-hour window of ground motion. Station channels verified against IRIS availability — UW network (Camp Muir, Carbon, White River, Tahoma Creek, Puyallup Valley) and CC Cascade Chain network (Paradise, Crystal Mtn, Nisqually). Seismicity data from the USGS Earthquake Hazards API.
